Java Toggle Button Implementation

This page documents a simple toggle button implementation for a Minecraft mod screen. Below are sections for code, notes, and customizations.

Main Code Example


// Calculate center position for the toggle button
int centerX = (this.width - 35) / 2;
int centerY = (this.height - 27) / 2;

// Create and configure the toggle button
toggleButton = new Toggle(centerX, centerY);
toggleButton.configure(
    "testing_toggle",
    () -> {
        System.out.println("Toggle is on, switching screen.");
        Minecraft.getInstance().setScreen(new JoinMultiplayerScreen(this));
    },
    () -> {
        System.out.println("Toggle is off, staying on current screen.");
        Minecraft.getInstance().setScreen(this);
    }
);

this.addRenderableWidget(toggleButton);
